How It Works

The Technology Stack

Our system leverages the power of Java Spring Boot for robust backend logic and OpenCV for advanced computer vision capabilities. Face embeddings are stored securely in MySQL, ensuring high availability and data integrity.

  • Java 17 & Spring Boot 3
  • MySQL Database
  • OpenCV for Face Detection
  • Modern Responsive UI

System Workflow

1 User registers face via webcam.
2 System extracts facial features (embeddings).
3 Data is encrypted and saved to MySQL.
4 Admin starts attendance session.
5 Faces recognized & attendance logged automatically.
